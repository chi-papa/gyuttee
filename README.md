# 🍊 GYUTTEE

> 画像をギュッと圧縮・リサイズできるブラウザ完結ツール

**[▶ 使ってみる](https://chi-papa.github.io/gyuttee/)** （GitHub Pages）

---

## 特徴

- **ブラウザだけで動く** — サーバーへの送信なし、インストール不要
- **Before / After 比較** — スライダーを動かして元画像と圧縮後を並べて確認
- **品質スライダー** — 1〜100% でリアルタイムにプレビュー更新
- **リサイズ機能** — 幅・高さを指定してサイズ変更（縦横比ロック対応）
- **3形式に対応** — JPEG / PNG / WebP から選んで出力
- **ファイル情報の表示** — 元サイズ・圧縮後サイズ・削減率・解像度を表示

---

## 使い方

1. 画像をドロップ、またはクリックしてファイルを選ぶ
2. 出力形式（JPEG / PNG / WebP）を選ぶ
3. 品質スライダーで圧縮率を調整
4. 必要であれば幅・高さを変更してリサイズ
5. **ダウンロード** ボタンで保存

---

## 技術情報

| 項目 | 内容 |
|------|------|
| 動作環境 | モダンブラウザ（Chrome / Edge / Firefox / Safari） |
| 依存ライブラリ | なし（素のHTML + CSS + JavaScript） |
| ファイル構成 | 単一HTMLファイル（`gyuttee.html`） |
| 画像処理 | Canvas API (`toDataURL`) |

### 圧縮の仕組み

- **JPEG / WebP** — `canvas.toDataURL('image/jpeg', quality)` で品質を指定して圧縮
- **PNG** — Canvas で再描画（可逆圧縮のため品質スライダーは無効）
- **リサイズ** — Canvas のサイズを変更して再描画
